Module Shuwarin.Style

Element styling.

Style is basically alters how your element look like. You can change it's border, font, colors, ...

Usage:

  • -- Modify element style: Set background color to red.
-- Shuwarin use color in 0..1 range.
element.style:setBackgroundColor({1, 0, 0, 1})

  • -- Modify element style: Add box shadow and blur to the shadow.
-- The shadow will be placed with offset 1px horizontal and 2px vertical
-- and with 3px blur strength. Also set the color to black with 90% alpha.
element.style:setBoxShadow(1, 2, 3, {0, 0, 0, 0.9})

Shuwarin.Style:setBoxShadow (offx, offy, blur, color)
Set box shadow properties. Calling this function without parameters disables box shadow completely.

Parameters:

  • offx number Offset X position of shadow.
  • offy number Offset Y position of shadow.
  • blur number Blur radius in pixels. Set to 0 to disable blur.
  • color table Shadow color, with 4 numbers inside (index 1, 2, 3, & 4). Defaults to black if none specified.
