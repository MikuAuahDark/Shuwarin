Module Shuwarin.Elements.Text

Simple text display.

This module handles text display. Line break is handled automatically. Please note that to set the text color from style, use Shuwarin.Style:setColor and to set the text shadow, use Shuwarin.Style:setBoxShadow instead of Shuwarin.Style:setTextShadow.

text:init (txt, newlinetospace)
Create new text display. You can pass simple text, or colored text. If you want colored text, please see Text:addf.

Parameters:

  • txt string or table Text to show.
  • newlinetospace boolean Convert new line into space? (only if txt is string)

Returns:

    Shuwarin.Elements.Text Text object.

Text display update. Unlike other elements, this method can be called by user code if needed.
