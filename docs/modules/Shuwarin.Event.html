Module Shuwarin.Event

Event handling mechanism

Events are needed to handle user input. This can range from touch events to keyboard input (textinput). For touch position, the position is always reported relative to current element.

Shuwarin.Event:addHandler (ev, func)
Add new event handler. Event handler function returns true if the event should not forwarded to next event handler, or false if the event should be forwarded to next event handler.

Parameters:

  • ev string Event name to handle.
  • func function Event handler function callback.

Remove event handler.

Parameters:

  • ev string Event name to handle.
  • func function Event handler function callback to remove.

Returns:
