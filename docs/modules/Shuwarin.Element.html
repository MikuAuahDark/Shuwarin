Module Shuwarin.Element

Base element class.

Element, can be called widgets is the base objects which shows the actual UI in Shuwarin. All elements like button, text input, checkbox, and other are derived from this class. This class can't be constructed directly.

Functions

Shuwarin.Element:update (deltaT)
Update element state. You should override this function when defining your own drawing function.

Parameters:

  • deltaT number Time passed since last frame in seconds.
Shuwarin.Element:_internalUpdate (deltaT)
Internally update element state. This is wrong method that you should override. Override Shuwarin.Element:update instead!

Parameters:

  • deltaT number Time passed since last frame in seconds.
Shuwarin.Element:draw ()
Draw current element. You should override this function when defining your own drawing function.
Shuwarin.Element:_internalDraw ()
Internally draw current element. You should not override this function as this function ensure style ordering. Only override if you really know what are you doing.
Shuwarin.Element:setPosition (x, y)
Set element position.

Parameters:

  • x number X position of element relative to current attached layout.
  • y number Y position of element relative to current attached layout.
